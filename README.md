# QuickRecon - Automated Recon Tool for VAPT

QuickRecon is a lightweight reconnaissance automation tool written in Python.  
It helps penetration testers and bug bounty hunters quickly gather information about a target domain during the initial reconnaissance phase.

---

## Features

- Subdomain Enumeration
- DNS Resolution
- Basic Port Scanning
- HTTP Header Analysis
- Technology Detection
- Multithreaded Subdomain Scanning
